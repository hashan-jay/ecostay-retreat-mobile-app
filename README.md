EcoStay Retreat Mobile Application

Overview:
EcoStay Retreat is a Mobile Application designed to simplify eco-friendly resort bookings and enhance user experience by promoting sustainable tourism. The app allows users to explore eco-retreats, make reservations, and manage their booking effectively.

Features:
1) Browse eco-friendly retreats and accommodations
2) Real-time booking and reservation system
3) Search and filter retreats based on location and preferences
4) User account management (Register/Login)
5) Secure booking process
6) User-friendly mobile interface

Technologies used:
1) Java (Android development)
2) Android Studio
3) SQLite (Database)
4) XML (UI design)

Getting started:
1) Clone the Repository
https://github.com/hashan-jay/ecostay-retreat-mobile-app.git
2) Open in Android Studio
- Open Android Studio
- Click "Open Project"
- Select the cloned folder
3) Run the Application
- Connect an Android device or start and emulator
- Click Run

Authentication:
Users can create accounts and log in securely to manage their bookings and preferences.

Sustainability Focus:

- Awareness of environmental conservation
